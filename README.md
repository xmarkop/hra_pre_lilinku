## Hra pre Lilinku

A simple browser Sokoban-inspired game built with HTML, CSS, and JavaScript.

### How to run

1. Open `index.html` in a browser.
2. Use the arrow keys or `WASD` to move the player.
3. Push all boxes onto the goal tiles to complete each level.

### Files

- `index.html` — game UI and layout
- `style.css` — visual styling
- `script.js` — game logic and rendering
